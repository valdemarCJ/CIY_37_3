# React Vite Bootstrap - Movie App

En simpel React webapp med JavaScript, Vite og Bootstrap der viser film fra et API.

## Funktioner
- Henter film data fra API: `http://localhost:5199/api/Movies?page=1&pageSize=20`
- Viser filmene i en Bootstrap tabel med kolonnerne: ID, Title, Year, Genre, Director
- Loading state og error handling

## Kør projektet
```bash
npm install
npm run dev
```

Applikationen kører på: http://localhost:5173/

## Teknologier
- React 19
- Vite 7
- Bootstrap 5
